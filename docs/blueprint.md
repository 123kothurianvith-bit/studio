# **App Name**: Snapter Games

## Core Features:

- Game Listing: Display a curated list of available games with details such as title, platform, price, and brief description.
- Search and Filter: Allow users to search for games by title and filter based on platform, genre, price range, and other relevant criteria.
- Wishlist: Enable users to save games to a wishlist for future purchase.
- AI Game Description Generation: When adding new games, admins can use an AI tool to generate engaging and informative game descriptions, including key features, gameplay mechanics, and target audience, for creating game descriptions automatically.

## Style Guidelines:

- Primary color: Dark Indigo (#4B0082) to evoke a sense of sophistication and depth, and to capture a sense of space.
- Background color: Very dark gray (#1E1E1E) to provide a modern, immersive experience.
- Accent color: Bright Violet (#EE82EE) to highlight interactive elements and calls to action.
- Body and headline font: 'Inter', a grotesque-style sans-serif.
- Use minimalist, vector-based icons for categories, platforms, and user interface elements.
- Employ a clean, grid-based layout to showcase games effectively.
- Use subtle animations, such as fade-in effects or hover states, to enhance the user experience.
